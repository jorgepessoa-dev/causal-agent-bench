# Code of Conduct

This project follows the [Contributor Covenant, v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be kind, be precise, assume good faith, and prioritize the scientific integrity of the benchmark over any individual contribution.

## Enforcement

Report concerns to the project owner: `utilitylabs.dev@gmail.com`. Reports are handled confidentially.

## Scope

Applies to project spaces (issues, PRs, discussions) and public spaces when representing the project (conferences, mailing lists, etc.).
